SimpleClockWidget Privacy Policy

Last updated: May 1, 2026

English 日本語

English

Information We Collect

SimpleClockWidget is an app that displays the current time and date on the Windows desktop.

The current version does not collect personally identifiable information, usage analytics, crash reports, location information, contacts, or file contents.

Information Stored on Your Device

The app stores only the following settings locally for the current Windows user:

  • Widget position
  • Widget size
  • Background color
  • Background transparency
  • Time and date text colors
  • Display language

Settings are stored at %LocalAppData%\ClockWidget\clock_widget.ini.

Data Transmission

The current version does not transmit saved settings or usage information to external servers.

Startup

SimpleClockWidget starts with Windows only if the user enables that option. Startup can be disabled by the user.

Contact

For questions about this app, please use the support information in Microsoft Store or the GitHub repository Issues.

日本語

収集する情報

SimpleClockWidget は、時刻と日付を Windows デスクトップに表示するためのアプリです。

現在のバージョンでは、個人を識別する情報、利用状況、分析データ、クラッシュレポート、位置情報、連絡先、ファイル内容を収集しません。

端末内に保存する情報

次の設定だけを、現在の Windows ユーザーの端末内に保存します。

  • ウィジェットの表示位置
  • ウィジェットのサイズ
  • 背景色
  • 背景の透過率
  • 時刻と日付の文字色
  • 表示言語

保存先は %LocalAppData%\ClockWidget\clock_widget.ini です。

外部送信

現在のバージョンでは、保存した設定や利用情報を外部サーバーへ送信しません。

自動起動

ユーザーが有効にした場合だけ、Windows 起動時に SimpleClockWidget を起動します。自動起動はユーザーが無効化できます。

お問い合わせ

このアプリに関するお問い合わせは、Microsoft Store のサポート情報または GitHub リポジトリの Issues からご連絡ください。
