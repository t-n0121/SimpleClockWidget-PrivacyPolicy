English

Information We Collect

SimpleClockWidget is an app that displays the current time and date on the Windows desktop.

The current version does not collect personally identifiable information, usage analytics, crash reports, location information, contacts, or file contents.

Information Stored on Your Device

The app stores only the following settings locally for the current Windows user:

  • Widget position and size
  • Background color and opacity
  • Time and date text colors
  • Display language
  • Time zone and date format
  • Saved theme settings

Settings are stored at %LocalAppData%\ClockWidget\clock_widget.ini.

Data Transmission

The current version does not transmit saved settings, usage information, or personal information to external servers.

Startup

SimpleClockWidget starts with Windows only if the user enables that option. Startup can be disabled by the user.

Contact

For questions about this app, please use the support information shown in Microsoft Store.

日本語

収集する情報

SimpleClockWidget は、現在の時刻と日付を Windows デスクトップに表示するためのアプリです。

現在のバージョンでは、個人を識別する情報、利用状況、分析データ、クラッシュレポート、位置情報、連絡先、ファイル内容を収集しません。

端末内に保存する情報

このアプリは、現在の Windows ユーザーの端末内に次の設定のみを保存します。

  • ウィジェットの表示位置とサイズ
  • 背景色と透過率
  • 時刻と日付の文字色
  • 表示言語
  • タイムゾーンと日付形式
  • 保存したテーマ設定

設定は %LocalAppData%\ClockWidget\clock_widget.ini に保存されます。

外部送信

現在のバージョンでは、保存した設定、利用情報、個人情報を外部サーバーへ送信しません。

自動起動

ユーザーが有効にした場合のみ、Windows 起動時に SimpleClockWidget を起動します。自動起動はユーザーが無効にできます。

お問い合わせ

このアプリに関するお問い合わせは、Microsoft Store に表示されるサポート情報をご利用ください。
