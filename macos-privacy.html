English

Information We Collect

SimpleClockWidgetMac is a macOS utility that displays a small clock panel on the desktop and provides menu bar controls.

The current version does not collect personally identifiable information, usage analytics, crash reports, location information, contacts, calendar data, photos, microphone data, camera data, screen contents, or file contents.

Information Stored on Your Device

The app stores only non-sensitive preferences locally on your Mac, including:

  • Clock panel position
  • Clock scale and visibility
  • Language, time zone, and date format preference values
  • Clock color and theme preference values
  • Cat type and cat size preference values
  • Whether the first-run guide has been shown
  • Hourly cat animation setting
  • Always-on-top setting
  • Click-through setting
  • Launch at login setting

These preferences are stored using macOS app preferences for the app's bundle identifier, com.t-n0121.SimpleClockWidgetMac.

Data Transmission

The current version does not transmit saved settings, usage information, or personal information to external servers. The app does not include advertising SDKs, tracking SDKs, analytics SDKs, or third-party data collection services.

Permissions and Entitlements

App Sandbox is enabled. The app does not request network access, file access, Screen Recording, Accessibility, Camera, Microphone, Contacts, Calendars, Photos, Bluetooth, or Location permissions.

Launch at Login

SimpleClockWidgetMac starts at login only if the user enables that option in Settings. The setting can be disabled by the user, and macOS may also show or manage the login item in System Settings.

Data Sharing

No personal data is collected, sold, shared, or used for tracking.

Contact

For questions about this app, please use the support information shown on the App Store product page.

日本語

収集する情報

SimpleClockWidgetMac は、macOSのデスクトップに小さな時計パネルを表示し、メニューバーから操作できるユーティリティアプリです。

現在のバージョンでは、個人を識別する情報、利用状況、分析データ、クラッシュレポート、位置情報、連絡先、カレンダー情報、写真、マイク、カメラ、画面内容、ファイル内容を収集しません。

端末内に保存する情報

このアプリは、次のような非機密の設定のみをMac内に保存します。

  • 時計パネルの表示位置
  • 時計の拡大率と表示状態
  • 表示言語、タイムゾーン、日付形式の設定
  • 時計の色とテーマ設定
  • 猫の種類とサイズ設定
  • 初回操作ガイドを表示済みかどうか
  • 毎時の猫アニメーション設定
  • 常に手前に表示する設定
  • クリック透過の設定
  • ログイン時起動の設定

これらの設定は、アプリのBundle ID com.t-n0121.SimpleClockWidgetMac に紐づくmacOSのアプリ設定として保存されます。

外部送信

現在のバージョンでは、保存した設定、利用情報、個人情報を外部サーバーへ送信しません。広告SDK、トラッキングSDK、分析SDK、第三者のデータ収集サービスは含まれていません。

権限とEntitlement

App Sandboxは有効です。ネットワークアクセス、ファイルアクセス、画面収録、アクセシビリティ、カメラ、マイク、連絡先、カレンダー、写真、Bluetooth、位置情報の権限は要求しません。

ログイン時起動

ユーザーが設定画面で有効にした場合のみ、ログイン時にSimpleClockWidgetMacを起動します。この設定はユーザーが無効にできます。また、macOSのシステム設定にもログイン項目として表示または管理される場合があります。

データ共有

個人データの収集、販売、共有、トラッキング目的での利用は行いません。

お問い合わせ

このアプリに関するお問い合わせは、App Storeの商品ページに表示されるサポート情報をご利用ください。
